Overview

Provides support to add binary attachments to crash reports and feedback messages

This is used by [BITCrashManagerDelegate attachmentForCrashManager:], [BITFeedbackComposeViewController prepareWithItems:] and [BITFeedbackManager showFeedbackComposeViewWithPreparedItems:]

The attachment data as NSData. The instance will be ignore if this is set to nil!

The content type of your data as MIME type. If nil will be set to “application/octet-stream”
